- **项目地址**：https://github.com/jeremylong/DependencyCheck.git
- **版本**：12.0.2-SNAPSHOT
- **操作**：
  - 使用 `git clone` 克隆项目到本地
  - 执行 `mvn clean install -DskipTests "-Dspotbugs.skip=true"` 编译项目
- **遇到的问题及解决**：
  - ❌ **问题1**：SpotBugs Maven Plugin 报错 `Java returned: 1`
    - **原因**：本地 JDK 版本为 26，而 SpotBugs 4.8.6.6 不支持 JDK 26
    - **解决**：添加参数 `-Dspotbugs.skip=true` 跳过 SpotBugs 检查
  - ❌ **问题2**：PowerShell 中执行 Maven 命令报错 `Unknown lifecycle phase ".skip=true"`
    - **原因**：PowerShell 对 `-D` 参数解析方式不同
    - **解决**：用双引号包裹参数 `"-Dspotbugs.skip=true"`
- **结果**：✅ 编译成功，生成 `dependency-check-core-12.0.2-SNAPSHOT.jar`
